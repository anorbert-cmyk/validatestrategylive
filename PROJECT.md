# ValidateStrategy (formerly Rapid Apollo / Aether Logic)

## Vision
ValidateStrategy is a strategic UX analysis platform that uses AI (Perplexity API) to validate digital product ideas. It offers three tiers of analysis:
- **Observer**: Quick validation (1-part analysis).
- **Insider**: Strategic roadmap (2-part analysis).
- **Syndicate**: Complete UX strategy (6-part analysis with Figma prompts).

The platform aims to reduce the 70% failure rate of digital products by providing research-backed insights and actionable strategies.

## Key Features
- **AI Analysis**: Multi-turn integration with Perplexity API.
- **Tiers**: Observer, Insider, Syndicate.
- **Payments**: Stripe, LemonSqueezy, NOWPayments (Crypto).
- **Authentication**: MetaMask (Admin), Magic Link (Users).
- **Output**: Real-time streaming, Markdown rendering, PDF export.
- **Marketing**: Email nurturing (Resend), Demo gates, SEO optimized.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Radix UI, Framer Motion.
- **Backend**: Node.js (Express), Drizzle ORM (MySQL/SQLite), tRPC (implied/used).
- **Testing**: Vitest.
- **Infrastructure**: Railway (implied), Docker (implied).
