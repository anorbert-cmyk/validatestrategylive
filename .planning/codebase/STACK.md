# Tech Stack

## Core
- **Language**: TypeScript
- **Runtime**: Node.js
- **Frontend Framework**: React 19 (Vite)
- **Backend Framework**: Express
- **ORM**: Drizzle ORM
- **Database**: MySQL (using `mysql2` driver) / SQLite (inferred from some files, but `mysql2` is in deps)

## UI/UX
- **Styling**: Tailwind CSS 4
- **Components**: Radix UI
- **Animations**: Framer Motion, Tailwind Animate
- **Icons**: Lucide React

## Integrations
- **AI**: Perplexity API
- **Payments**: Stripe, LemonSqueezy, NOWPayments
- **Email**: Resend
- **Auth**: MetaMask (Admin), Magic Links

## Testing
- **Runner**: Vitest
